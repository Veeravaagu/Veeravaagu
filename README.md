## Featured Projects

### [Taco-DB — Disk-Based Relational Database Engine](https://github.com/veeravaagu-murugan/taco-db)
> C++17 · B+ Tree · Buffer Pool Manager · Volcano Execution Model · External Merge Sort

A disk-oriented relational database engine built in C++17 from the ground up.

- **Storage layer:** Buffer pool manager operating over fixed-size 4KB pages with LRU eviction under strict memory constraints
- **Indexing:** B+ Tree supporting O(log n) point lookups and range scans on datasets exceeding available memory, with correct split/merge logic
- **Query execution:** Volcano iterator model implementing selection, aggregation, external merge sort, and join algorithms (BNLJ + index nested loop)
- **Validation:** 100+ automated tests and TPC-H benchmark workloads confirming correctness across storage, indexing, and execution subsystems
